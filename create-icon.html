Twitch Redeem Overlay Icon Generator

This tool generates a simple application icon for your Twitch Redeem Overlay app.

Click the button to generate and download the icon. You'll need to save it as:

  • Windows: src/assets/app-icon.ico (may need conversion)
  • macOS: src/assets/app-icon.icns (may need conversion)
  • Linux: src/assets/app-icon.png
